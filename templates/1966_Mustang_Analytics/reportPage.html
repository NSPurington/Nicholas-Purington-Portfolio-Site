This project consists of sending an HTTP request to the Ebay API and returning pertinent data about the 1966 Ford Mustangs sold on Ebay. I'm using an Azure Logic Apps to send the HTTP request and parse the response, and an Azure SQL Server database to store the result data. The Logic Apps runs on a weekly recurrence @ 7pm PST.

The analytics report was created in Microsoft Power BI and is updated with the latest data when the Logic Apps runs each week.
